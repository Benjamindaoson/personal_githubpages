# Product UX Upgrade

## Why

The current site has a sound publishing foundation, but it still reads like a documentation site. The next product step is to make the first-screen experience, learning path, task bank, and lesson pages behave like a complete AI learning product.

## What Changes

- Replace the basic homepage with an AI learning product entry that exposes the course promise, learning paths, active units, task bank, and teacher reuse model.
- Add a dedicated roadmap page for Large Models, RAG, and Agents.
- Add a task bank page for labs, projects, assessments, and teacher reuse assets.
- Add reusable lesson chrome components so unit pages can consistently show objectives, prerequisites, estimated time, status, formats, and CTA links.
- Extend unit metadata so the curriculum source can drive product UI, future search, progress, and agent personalization.

## Out Of Scope

- User accounts and persistent progress tracking.
- Paid membership or commerce.
- Real AI assistant runtime.
- Plugin ecosystem for LeetCode, Chrome, VS Code, or IDEs.
- Full visualizer implementation.
